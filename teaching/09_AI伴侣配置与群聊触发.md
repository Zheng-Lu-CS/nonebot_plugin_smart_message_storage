# AI 伴侣配置与群聊触发

本文对应本地机器人项目：

```text
D:\WorldModel\bot\smart_message_bot
```

主要修改两个文件：

```text
D:\WorldModel\bot\smart_message_bot\.env.prod
D:\WorldModel\bot\smart_message_bot\plugins\ai_companion.py
```

改完 `.env.prod` 后必须重启机器人。

## 1. 自定义背景信息

背景、人设、关系设定、说话风格都写在：

```env
AI_COMPANION_PERSONA="你的名字是“苏语桐”，你是一名在电子科技大学读书的大一女生。你说话温柔、自然、简短，像 QQ 私聊，不要输出 [表情]、[图片] 这类占位符。"
```

建议把要求写具体：

- 她叫什么。
- 她和你的关系是什么。
- 她了解你哪些背景。
- 她说话应该短一点还是详细一点。
- 禁止输出什么内容。

如果人设很长，尽量保持整个值在同一对英文双引号里。多行也可能可用，但更容易因为引号或换行写错。

## 2. 私聊白名单

只有这些 QQ 号私聊机器人时，AI 伴侣才会回复：

```env
AI_COMPANION_PRIVATE_USER_IDS=1710268315
```

多个 QQ 用英文逗号分隔：

```env
AI_COMPANION_PRIVATE_USER_IDS=1710268315,123456789
```

`AI_COMPANION_TARGET_USER_ID` 会自动加入私聊白名单。

## 3. 群聊白名单

只有这些群允许触发 AI 伴侣：

```env
AI_COMPANION_GROUP_IDS=983881028
```

多个群号用英文逗号分隔：

```env
AI_COMPANION_GROUP_IDS=983881028,123456789
```

普通群消息不会自动回复，必须满足下面任意一种触发方式。

## 4. 群聊触发名

当前支持两种触发：

```text
@机器人 你在吗
@苏语桐 你在吗
```

其中 `@机器人` 是 QQ 真正艾特机器人账号；`@苏语桐` 是文本触发名。

触发名在这里改：

```env
AI_COMPANION_TRIGGER_NAMES=苏语桐
```

多个名字用英文逗号分隔：

```env
AI_COMPANION_TRIGGER_NAMES=苏语桐,小桐,语桐
```

这样群里发送这些消息都会触发：

```text
@苏语桐 你在吗
小桐 今天吃什么
语桐帮我看看这张图
```

注意：这只是文本匹配，不会真的改变机器人 QQ 昵称。想让 QQ 的真实 @ 名也显示为“苏语桐”，需要在 QQ 群里给机器人账号改群昵称。

## 5. 回复延迟

从收到消息到发送回复之间，可以设置一个随机延迟：

```env
AI_COMPANION_REPLY_DELAY_MIN_SECONDS=1
AI_COMPANION_REPLY_DELAY_MAX_SECONDS=4
```

含义是：生成回复后，再随机等待 1 到 4 秒发送。

如果想立刻回复：

```env
AI_COMPANION_REPLY_DELAY_MIN_SECONDS=0
AI_COMPANION_REPLY_DELAY_MAX_SECONDS=0
```

如果想更像真人一点：

```env
AI_COMPANION_REPLY_DELAY_MIN_SECONDS=3
AI_COMPANION_REPLY_DELAY_MAX_SECONDS=10
```

## 6. 图片和表情

图片理解开关：

```env
AI_COMPANION_ENABLE_IMAGE_INPUT=true
AI_COMPANION_MAX_IMAGES_PER_MESSAGE=3
```

当你私聊或在白名单群里触发她，并且消息里带图片时，插件会尽量从 OneBot 消息里取得图片 URL，然后把图片一起发给模型理解。

要求：

- `AI_COMPANION_MODEL` 使用支持视觉输入的模型。
- 你的 API 服务商支持 Chat Completions 的图片输入格式。
- NapCat/OneBot 能在消息段里提供可访问的图片 URL。

表情和表情包目前不会真的转成图片识别，但插件会告诉模型“用户发来了一个 QQ 表情/表情包”，让模型结合上下文理解情绪。

## 7. 避免输出 `[表情]`

插件已经做了两层处理：

- 系统提示里要求模型不要输出 `[表情]`、`[图片]`、`[CQ:...]`。
- 发出前会自动清理这些占位符。

如果仍偶尔出现，继续在人设里强调：

```env
AI_COMPANION_PERSONA="...不要输出 [表情]、[图片]、[CQ:...] 这类占位符；想表达情绪就直接用自然语言。"
```

## 8. 重启方式

先确认没有多个 `bot.py` 同时运行。然后只用虚拟环境启动：

```powershell
cd D:\WorldModel\bot\smart_message_bot
.\.venv\Scripts\Activate.ps1
python bot.py
```

启动成功后日志里应该看到：

```text
Loaded plugin "plugins.ai_companion"
OneBot V11 | Bot 2942432394 connected
```

私聊测试：

```text
你好呀
```

群聊测试：

```text
@苏语桐 你在吗
```

或真正艾特机器人：

```text
@机器人 你在吗
```

状态检查：

```text
/女友状态
```

状态里会显示私聊白名单、群聊白名单、群聊触发名、回复延迟和模型。
