# 任务驱动：导出一个群聊和一个私聊好友

目标：用 QQ 小号接入机器人，采集一个群聊和一个私聊好友的消息，完成图片识别，并导出 CSV/JSON。

## 1. 准备信息

你需要准备：

- 小号 QQ：登录 NapCat 的机器人账号。
- 目标群号：要归档的群聊 ID。
- 目标好友 QQ：要归档的私聊好友 ID。
- OpenAI API Key：用于图片识别。

记录下来：

```text
BOT_QQ=小号QQ
TARGET_GROUP_ID=目标群号
TARGET_FRIEND_QQ=目标好友QQ
```

## 2. 启动服务

启动 NoneBot：

```powershell
Set-Location D:\WorldModel\bot\smart_message_bot
.\.venv\Scripts\Activate.ps1
python bot.py
```

启动 NapCat：

- 登录小号。
- 确认 OneBot v11 反向 WebSocket 连接到 `ws://127.0.0.1:8080/onebot/v11/ws`。
- 确认 NoneBot 日志显示连接成功。

## 3. 采集群聊

让小号加入目标群。

在群里发送几条测试消息：

```text
群聊归档测试 001
```

发送一张图片，再发送一句后文：

```text
这张图用于测试群聊图片识别
```

如果是从现在开始归档，保持机器人在线即可。

如果要补录已有群聊历史：

- 推荐把关键历史文本逐条复制或转发到小号能看到的群/私聊。
- 图片尽量发原图。
- 如果只能截图，就把聊天截图发给小号，让 AI 总结截图内容。
- 不推荐 QQ 合并转发聊天记录，因为当前插件不会展开合并转发。

## 4. 采集私聊好友

让目标好友和小号互发消息，或由你的个人号把关键历史补录给小号。

在小号与目标好友的私聊里发送：

```text
私聊归档测试 001
```

再发送一张图片和一句后文：

```text
这张图用于测试私聊图片识别
```

如果你想导出的是“个人号与好友之间已经存在的历史私聊”，小号不能直接读取这段历史。需要个人号把关键内容逐条复制/转发给小号，或发截图给小号。

## 5. 触发图片识别

在目标群里发送：

```text
/立即识别
```

在小号与目标好友的私聊里也发送：

```text
/立即识别
```

如果你是 `SUPERUSERS`，也可以发送：

```text
/立即识别 全部
```

建议先用当前会话的 `/立即识别`，便于排查。

## 6. 查询验证

在目标群里查：

```text
/查消息 群聊归档测试
```

或指定群号查：

```text
/查消息 目标群号 群聊归档测试
```

在目标好友私聊里查：

```text
/查消息 私聊归档测试
```

回复一条图片消息并发送：

```text
/识别
```

如果 `/查消息` 没有返回图片结果，先看 `06_故障排查.md` 的字体章节。

## 8. 导出 CSV 和 JSON

在 `D:\WorldModel\bot\smart_message_bot` 新建 `export_messages.py`：

```python
import csv
import json
import sqlite3
from pathlib import Path

DB_PATH = "qq_messages.db"
TARGET_GROUP_ID = 123456789
TARGET_FRIEND_QQ = 987654321

OUT_DIR = Path("exports")
OUT_DIR.mkdir(exist_ok=True)

COLUMNS = [
    "id",
    "time",
    "self_id",
    "user_id",
    "group_id",
    "sender_nickname",
    "sender_card",
    "message_id",
    "reply_id",
    "raw_message",
]


def fetch_rows(where_sql, params):
    con = sqlite3.connect(DB_PATH)
    con.row_factory = sqlite3.Row
    try:
        sql = f"""
        select {", ".join(COLUMNS)}
        from group_messages
        where {where_sql}
        order by id asc
        """
        return [dict(row) for row in con.execute(sql, params).fetchall()]
    finally:
        con.close()


def write_csv(path, rows):
    with open(path, "w", newline="", encoding="utf-8-sig") as f:
        writer = csv.DictWriter(f, fieldnames=COLUMNS)
        writer.writeheader()
        writer.writerows(rows)


def write_json(path, rows):
    with open(path, "w", encoding="utf-8") as f:
        json.dump(rows, f, ensure_ascii=False, indent=2)


group_rows = fetch_rows("group_id = ?", (TARGET_GROUP_ID,))
friend_rows = fetch_rows("group_id = -1 and user_id = ?", (TARGET_FRIEND_QQ,))

write_csv(OUT_DIR / f"group_{TARGET_GROUP_ID}.csv", group_rows)
write_json(OUT_DIR / f"group_{TARGET_GROUP_ID}.json", group_rows)

write_csv(OUT_DIR / f"private_{TARGET_FRIEND_QQ}.csv", friend_rows)
write_json(OUT_DIR / f"private_{TARGET_FRIEND_QQ}.json", friend_rows)

print(f"group rows: {len(group_rows)}")
print(f"private rows: {len(friend_rows)}")
print(f"exported to: {OUT_DIR.resolve()}")
```

把这两行改成你的真实 ID：

```python
TARGET_GROUP_ID = 123456789
TARGET_FRIEND_QQ = 987654321
```

运行：

```powershell
python export_messages.py
```

导出文件在：

```text
D:\WorldModel\bot\smart_message_bot\exports\
```

包括：

```text
group_<群号>.csv
group_<群号>.json
private_<好友QQ>.csv
private_<好友QQ>.json
```

## 9. 验收标准

完成后应满足：

- 目标群有文本消息入库。
- 目标好友私聊有文本消息入库。
- 至少一张群聊图片被识别并回写为 `[image:{summary:"...",tip:"..."}]`。
- 至少一张私聊图片被识别并回写。
- `/查消息` 能返回结果图片。
- CSV/JSON 文件生成，行数与数据库查询一致。
